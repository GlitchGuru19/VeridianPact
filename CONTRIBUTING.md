# Contributing to VeridianPact

Thank you for your interest in contributing to the VeridianPact project! We welcome contributions from everyone.

## How to Contribute
1. **Fork the Repository**: Click the 'Fork' button on the top right of the repo page.
2. **Clone Your Fork**: Use `git clone` to clone your fork to your local machine.
3. **Create a Branch**: Use `git checkout -b your-branch-name` to create a new branch.
4. **Make Changes**: Make your changes in your local repository.
5. **Commit Changes**: Use `git commit -m "Your commit message"` to commit your changes.
6. **Push to Your Fork**: Use `git push origin your-branch-name` to push your changes.
7. **Create a Pull Request**: Navigate to the original repo and click the 'New Pull Request' button.

## Guidelines
- Ensure your code adheres to the project's coding style. 
- Write clear commit messages.
- Test your changes before submitting.

## Reporting Issues
If you encounter any issues, please use the GitHub issues tracker to report them.

## Questions?
Feel free to reach out to the maintainers for any questions or clarifications.
